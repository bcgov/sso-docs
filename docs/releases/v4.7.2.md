# CSS v4.7.2
2026-01-08

### New Features 🎉
- Removed Keycloak managed email_verified from the default mapper list for all IDPs. For BCSC that supplies their own email_verified, the upstream value will be passed through as it is. We made this change because the Keycloak managed email-verified has not been providing a functional value that reflects the upstream IDP value.

### Bug fixes and enhancement 🐛
- Enhanced client secret update message to specify the new secret download location.
- Enhanced CSS API username search with exact match. This applies to endpoints with username as the query parameter such as [BCeID users search](https://api.loginproxy.gov.bc.ca/openapi/swagger/#/Users/get_integrations__integrationId___environment__bceid_users).
- Fixed an issue for SSO Admin Dashboard. The dashboard ran into an infinite loop in certain circumstances.
