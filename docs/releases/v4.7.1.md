# CSS v4.7.1
2025-12-04

### New Features 🎉
- Added One Time Passcode (OTP) as a zero-assurance log-in option for applications. This feature is developed, and the pilot is deployed into DriveBC. We are collecting user feedback and setting up onboarding protocol before a broader release. Please contact SSO team if you want to know more about this log-in option. 

### Bug fixes and enhancement 🐛
- Fixed an issue for the field Additional Role Attribute (optional). Now this field is visible only if the selected Identity Provider(s) supports role assignment.
- Fixed an issue for SSO Admin Dashboard. The dashboard ran into an infinite loop in certain circumstances.
