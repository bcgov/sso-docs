# CSS v4.14.0
2026-06-02

### New Features 🎉
- For IDIR and IDIR - MFA integrations, user searches for role assignment now seamlessly return results from the respective identity providers (if users are not already in the SSO system). Previously, IDIR required an additional lookup tool to load users into the SSO system, while IDIR - MFA had no lookup capability.
- New BCeID integrations now require separate approval from the Identity and Information Management (IDIM) team for each selected environment.

### Bug fixes and enhancement 🐛
- Replaced IDP GUID with Username as the search criterion for IDIR and IDIR - MFA in the Assign Users to Roles tab.

### Miscellaneous 🧩
- Updated SSO team contact information from Rocket.Chat to Microsoft Teams. The Rocket.Chat #sso channel is being decommissioned. Please contact the team via the [Microsoft Teams Keycloak How-to Channel](https://teams.microsoft.com/l/channel/19%3A35d0b3389e39479590ba45a19a67a3ba%40thread.tacv2/SSOKeycloak-howto?groupId=a80418da-c27b-406e-89ab-7695b61924d8&tenantId=6fdb5200-3d0d-4a8a-b036-d3685e359adc) or by [email](mailto:bcgov.sso@gov.bc.ca).
